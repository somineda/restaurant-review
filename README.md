# 🍱 솜슐랭 맛집 리뷰 플랫폼

> 사용자가 솜슐랭에 등록된 식당 상세 정보를 보고 리뷰를 작성하거나 다른 사람의 후기를 볼 수 있는 웹서비스
> Django + Django REST Framework 기반으로 구축되었으며 JWT 인증과 관리자 페이지도 지원합니다.
